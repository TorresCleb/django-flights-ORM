# ✈️ Sistema de gerenciamento de voos
### Projeto desenvolvido para o curso CS50's Web Programming with Python and JavaScript (Harvard University)

Este projeto foi desenvolvido como parte do meu aprendizado no curso **CS50's Web Programming with Python and JavaScript** da **Harvard University**. 

O objetivo principal é explorar as capacidades do Django ORM (Object-Relational Mapper), manipulação de bancos de dados relacionais e a criação de interfaces dinâmicas com o sistema de templates do Django.

## 📝 Sobre o Projeto

O sistema simula o gerenciamento de uma companhia aérea, permitindo cadastrar aeroportos e associar voos a eles, garantindo a integridade dos dados através de chaves estrangeiras.

### Principais Aprendizados:
- **Modelagem de Dados:** Criação de tabelas e relacionamentos Many-to-One (Foreign Keys).
- **Django Migrations:** Entendimento do fluxo de `makemigrations` e `migrate`, além de resolução de erros de integridade.
- **Django Shell:** Manipulação e consulta de dados diretamente via linha de comando.
- **Templates:** Uso de herança de templates com `{% extends %}` e `{% block %}` para criar um layout reutilizável.

## 🛠️ Tecnologias Utilizadas

- **Python 3**
- **Django Framework**
- **SQLite** (Banco de dados)
- **HTML/CSS**
